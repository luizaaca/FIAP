# Estudos de IA para Desenvolvimento de Software

Este repositório contém os estudos realizados durante minha pós-graduação em Inteligência Artificial aplicada ao desenvolvimento de software.

> ⚠️ Este repositório será atualizado continuamente conforme novos conteúdos forem vistos e praticados no curso. Ele serve como espaço para experimentos e estudos práticos em IA e desenvolvimento de software.

## Estrutura do Projeto

- **ML/**: Algoritmos de Machine Learning clássicos, exemplos didáticos e scripts de regressão, classificação e clustering.
	- `regressao_linear.py`: Demonstração de regressão linear, testes com funções lineares e quadráticas, visualização dos resultados.
	- `classificador_textos_naive_bayes.py`: Exemplo de classificação de textos usando Naive Bayes.
	- `docs/`: Documentação didática em Markdown sobre algoritmos como Árvore de Decisão, KMeans, KNN, Naive Bayes e Regressão Linear.

- **DL/**: Exemplos de Deep Learning com Keras/TensorFlow.
	- `rede_neural_simples.py`: Rede neural simples para regressão, com explicações detalhadas e validação em novos dados.
	- `rede_neural_simples_2.py`: Rede neural para classificação binária, com comentários didáticos.

## Objetivo

O objetivo deste repositório é consolidar o aprendizado prático e teórico em IA, com foco em:
- Implementação de algoritmos clássicos de ML e DL
- Explicações didáticas e exemplos comentados
- Testes de generalização e análise de resultados
- Documentação para consulta e revisão

Consulte os arquivos de código e a documentação em `ML/docs/` para aprofundar nos conceitos e exemplos.
